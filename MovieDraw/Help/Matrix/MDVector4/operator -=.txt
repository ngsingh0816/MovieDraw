<name>operator -=</name>
<type>
	<name>Function</name>
	<return>
		<type>MDVector4</type>
		<desc>Returns the two vectors subtracted from each other.</desc>
	</return>
	<argument0>
		<type>MDVector4</type>
		<name>p</name>
		<desc>The vector to subtract from the other vector.</desc>
	</argument0>
</type>
<desc>Subtracts two vectors and sets the current vector to the difference.</desc>
<example>
MDVector4 vector = MDVector4Create(5, 0, 1, 2);
MDVector4 vector2 = MDVector4Create(1, 3, 1, 3);
vector -= vector2;		// Gives the vector (4, -3, 0, -1)
</example>
